Thank you for checking out Zhouzi (pronounced 'CHO-z…ôh')

Zhouzi is a Vue data table component, utilizing Vuex

With Zhouzi, I wanted to create a table component that doesn't use the actual table element, but rather, take advantage of the native CSS2 display: table-* layout mode.
